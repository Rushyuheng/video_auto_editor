# Auto video editing 
###### tags: side_project

## require environment
* python 3.7up
* setuptool
* pyqt5
* pillow
* moviepy

run command  
```=bash
sudo pip install setuptool  
sudo pip install pyqt5
sudo pip install pillow
sudo pip install moviepy
```
execute main.py
```=bash
python ./main.py
```
